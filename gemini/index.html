User Context & Evaluation Criteria

This dashboard translates the complex EEE/EECE program data of GST cluster universities into an actionable decision framework. It is specifically calibrated for a student (HSC 2025, Dhaka Board) aiming for a fully-funded MS abroad (Germany/Japan) in Robotics.

Weighting Logic Applied: Smaller universities with active research faculty, strong IEEE Student Branches, and a culture of graduate-level publication heavily outrank larger institutions with passive faculty. The data presented is a synthesized evaluation of public records, UGC reports, and IEEE activity metrics.

🔬 Research Weight 40% (Publications, PhD Ratios)
⚙️ Facilities Weight 25% (Robotics/IoT Labs)
🌐 Network Weight 20% (IEEE SB, Global Alumni)
📚 Curriculum Weight 15% (EEE vs EECE alignment)
